# 查看退款失败订单

## 请求URL

- 地址：`/business/order`

## 请求方式

- `POST`
- `application-json`

## 请求参数

| 参数名       | 必选 |  类型  | 说明             |
| :----------- | :--: | :----: | ---------------- |
| refundStatus |  是  | number | 固定为9-退款失败 |
| current      |  是  | number | 页码             |
| size         |  是  | number | 页大小           |


## 结果返回示例

> 同[订单列表分页查询](订单列表分页查询.md)的返回参数一致
